Modul ini untuk praktikum **Pemrograman Berbasis Web (PBW)** di SMP Mentari. Tujuannya untuk belajar membuat **CRUD aplikasi** menggunakan **Laravel 12** dan **Filament 4**, serta mengakses dashboard admin interaktif.  

Teknologi:
- PHP 8.2  
- Laravel 12.x  
- Filament 4.x  
- Composer & MySQL  

Cara Menjalankan:
1. Clone repo: `git clone <url>`  
2. Install dependencies: `composer install`  
3. Copy `.env.example` jadi `.env`, sesuaikan database  
4. Jalankan migrasi: `php artisan migrate --seed`  
5. Jalankan server: `php artisan serve`  
6. Buka `http://localhost:8000/admin`
